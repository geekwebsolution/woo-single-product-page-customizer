== Description ==

Using this smart plugin, allows you to add text or HTML to Woocommerce Single product page, without editing the theme and WooCommerce plugin!

Visual HTML editor allows a user to add HTML in different positions of a Single product page.
Below are listed all positions of the Single product page where you can show your custom HTML from the admin side, no need to customize the WooCommerce plugin.

== Installation ==

= Automatic Installation =

The best way to install the plugin on your WordPress platform is automatic installation. So let's start, first login to your admin dashboard. and click on the <strong>Plugins</strong> menu then click on the <strong>Add New</strong> option.

Now you have a plugin list page on your screen, here in the search box, search for the <strong>Woocommerce Single Product Page Customizer</strong>. Now look for our plugin and click on the <strong>Install</strong> button that's it.

<strong>Admin Dashboard > Plugins > Add New > </strong>Search for <strong>Woocommerce Single Product Page Customizer</strong> plugin.

= Manual Installation =

So let's talk about manual installation. Before you start download the plugin. Extract the .zip in the <strong>wp-content/plugins/</strong> directory of your WordPress installation and then activate the plugin from the Plugins page.

After activating the plugin, go to <strong>Woocommerce > Woocommerce Single Product Page Customizer</strong>.
